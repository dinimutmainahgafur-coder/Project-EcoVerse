# EcoVerse
Small Actions, Big Impact
EcoVerse adalah aplikasi mobile berbasis Flutter yang dirancang untuk meningkatkan kesadaran masyarakat, khususnya mahasiswa, dalam menerapkan gaya hidup ramah lingkungan melalui pendekatan gamifikasi. Aplikasi ini mengajak pengguna menyelesaikan berbagai misi harian, mempelajari materi edukasi lingkungan, serta memperoleh Eco Point, Badge, dan Level sebagai bentuk apresiasi atas setiap aksi positif yang dilakukan. Dengan konsep yang interaktif, EcoVerse diharapkan dapat membantu membangun kebiasaan ramah lingkungan secara berkelanjutan.

## Getting Started

### Prerequisites
- Flutter SDK 3.35+
- Dart SDK 3.0+
- Android Studio / VS Code

### Installation
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Fitur yang ada
- Daily Mission System
- Eco Points & Leveling
- Badge Collection
- Eco Journal
- Eco Academy
- Eco Challenge
- Leaderboard
- Profile Management

# EcoVerse

Small Actions, Big Impact

A gamified environmental awareness mobile application built with Flutter.

## Getting Started

### Prerequisites

- Flutter SDK 3.35+
- Dart SDK 3.0+
- Android Studio / VS Code

### Installation

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Features

- Daily Mission System
- Eco Points & Leveling
- Badge Collection
- Eco Journal
- Eco Academy
- Eco Challenge
- Leaderboard
- Profile Management

## SDGs

SDGs 12 - Responsible Consumption and Production

## Architecture

Clean Architecture with Provider state management.
